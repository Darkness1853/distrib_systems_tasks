# Checklist

## Чек-лист (OWASP Top 10)

- [x] **Injection**: Проверка на SQL/Command Injection.
- [x] **Broken Authentication**: Слабые пароли, отсутствие MFA.
- [x] **Sensitive Data Exposure**: Хранение паролей в открытом виде.
- [ ] **XML External Entities (XXE)**: Обработка XML.
- [x] **Broken Access Control**: Доступ к чужим данным по ID.
- [x] **Security Misconfiguration**: Дефолтные настройки, лишние порты.
- [x] **Cross-Site Scripting (XSS)**: Валидация ввода в веб-интерфейсе.
- [ ] **Insecure Deserialization**: Pickle/Yaml load.
- [x] **Using Components with Known Vulnerabilities**: Старые библиотеки.
- [x] **Insufficient Logging & Monitoring**: Отсутствие логов доступа.
